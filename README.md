# Keylogger Project

## Description
This project is a simple keylogger built using HTML, CSS, and JavaScript. It logs keypresses and displays them on the screen, with options to start and stop logging.

## Features
- Start and stop logging keypresses.
- Display logged keypresses in real-time.
- Simple and clean user interface.

### Installation
1. Clone the repository to your local machine:
   ```bash
   git clone https://github.com/Jenish-Patel31/keylogger-javascript-project.git
   cd keylogger-javascript-project
2.Open the Project1_keyblooger.html file in your web browser to run the keylogger.


## Running the Keylogger
1. Open Project1_keyblooger.html in a web browser.
2. Click the "Start Logging keypresses" button to begin logging keypresses.
3. Press keys on your keyboard. The keypresses will be logged and displayed on the screen.
4. Click the "Stop Logging keypresses" button to stop logging keypresses.
